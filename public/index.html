Bienvenido a la API de Juegos

Juegos

GET /api/games
Obtener todos los juegos

GET /api/games/:id
Obtener juego por ID

GET /api/games/name/:name
Obtener juego por Nombre

GET /api/games/name/filter/platform/:platform
Obtener juego por Plataforma

POST /api/games
Crear nuevo juego

PUT /api/games/:id
Actualizar juego

DELETE /api/games/:id
Eliminar juego
